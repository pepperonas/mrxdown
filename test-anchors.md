## Technische Details

Die Sprungmarken funktionieren durch:

1. Automatische ID-Generierung für Überschriften
2. Markdown Links mit `#` für interne Navigation
3. CSS-Styling für Anchor-Links
4. Smooth Scrolling im HTML-Export
